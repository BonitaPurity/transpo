# Frontend (TRANSPO HUB)

This package contains the Next.js frontend application.

## Local development
1. Create env file:
   ```bash
   copy .env.local.example .env.local
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

Open `http://localhost:3000`.

## Build checks
```bash
npm run lint
npm run build
```

For full project setup (backend + database + deployment), see the root [README.md](../README.md).
